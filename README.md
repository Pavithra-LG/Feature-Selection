# Feature-Selection
Feature Selection Study for Fraud Detection

This project analyzes the impact of feature selection on the performance of a Logistic Regression model for fraud detection.

1.Objective

To compare model performance using:

All features

SelectKBest-selected features

2.Methodology

Train Logistic Regression on full feature set

Apply SelectKBest (top 10 features)

Train model on reduced feature set

Compare recall for fraud class

3.Key Result

Feature selection reduces dimensionality while maintaining comparable recall for fraud detection.
